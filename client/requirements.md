## Packages
framer-motion | Essential for all requested smooth animations, sliding sidebar, and modal scaling
react-hook-form | Form state management
@hookform/resolvers | Zod validation integration for forms
zod | Schema validation
lucide-react | High-quality icons for medical theme and UI elements
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging tailwind classes without style conflicts

## Notes
- JWT Token is stored in localStorage ('medibot_token')
- Fetch requests use a custom wrapper to inject the Bearer token automatically
- Wouter handles routing
- All custom fonts (Plus Jakarta Sans, Inter) are imported in index.css
